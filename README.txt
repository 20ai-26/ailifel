AI LIFE 협찬 랜딩페이지 v2.0

주요 반영 사항
- 제공된 AI LIFE 공식 로고를 헤더, 메인, 푸터, 파비콘에 적용
- 협찬제품 1번(A타입 강아지) / 협찬제품 2번(B타입 고양이) 탭 전환 유지
- 제품 선택 버튼을 스크롤 중에도 쉽게 사용할 수 있도록 고정형으로 개선
- 제품 설명서 이미지를 데스크톱에서 원본 비율 전체 너비로 표시
- 모바일에서는 설명서를 축소하지 않고 좌우 스크롤로 확인
- 기존 협찬 혜택, 진행 순서, FAQ 및 모바일 최적화 유지

인스타그램 DM 버튼 링크는 index.html의 https://www.instagram.com/ 주소를 실제 계정 주소로 변경해 주세요.

[v2.0 버튼 수정]
협찬제품 선택 버튼의 고정(sticky) 동작만 제거했습니다. 버튼은 제품 선택 영역에만 표시되며 스크롤 시 제품 이미지를 따라다니거나 가리지 않습니다. 그 외 구성과 디자인은 v2.0 그대로 유지했습니다.

[v2.1 모바일 완성도 개선]
- PC 디자인과 구성은 v2.0 버튼 수정본 그대로 유지했습니다.
- 모바일에서 A/B 제품 사진 비율을 4:5로 통일하고 잘림을 제거했습니다.
- 제품 설명서는 모바일에서 좌우 스크롤 없이 읽을 수 있도록 7개 영역으로 나누어 세로 배치했습니다.
- 모바일 이미지 폭, 섹션 간격, 버튼 크기와 히어로 프레이밍만 조정했습니다.


[v2.2 업데이트]
- 최신 제품 설명서 이미지로 교체
- PC에서는 설명서 전체 이미지를 원본 비율로 표시
- 모바일에서는 좌우 이동 없이 읽을 수 있도록 5개 구간으로 세로 분할
- 그 외 레이아웃과 기능은 v2.1 그대로 유지


[v2.3 수정]
- 인스타그램 DM 신청하기 이미지/버튼 섹션 삭제
- 요청 문구 2줄로 교체
- 제품 색상 다음에 제품 설명서, 그 아래에 협찬 혜택이 오도록 순서 변경
- 그 외 내용은 수정하지 않음

[v2.3 모바일 설명서 정비]
- 기존 v2.3 레이아웃과 기능은 그대로 유지
- 모바일 설명서 크롭 경계를 콘텐츠 기준으로 재조정
- 오른쪽 설명 이미지에 섞이던 왼쪽 불필요 영역 제거
- 09·10·11 번호 배지와 상단 테두리가 잘리지 않도록 안전 여백 확보
- 모바일 가독성을 위해 분할 이미지를 고해상도로 재처리
